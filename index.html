Startup Page Clone Project

This project involved copying each element of a landing page, piece by piece. It taught the use of semantic html, and other basic html and css concepts.

The process was simple but effective: first, make a best effort attempt to copy the image, without viewing the source code. Second, guided instructions for completing the site, along with constant checking against the source.

By the end of the project, I had a simple, but complete, landing page.
